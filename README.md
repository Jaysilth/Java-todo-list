# Java To-Do List

A simple console-based To-Do List application written in Java.  
This project was built to practice Java fundamentals and GitHub.

## Features
- Add tasks
- View tasks
- Delete tasks
- Exit the application

## Technologies Used
- Java
- IntelliJ IDEA
- Git & GitHub

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run the `Main.java` file
4. Follow the menu options in the console

## What I Learned
- Java syntax and methods
- Using ArrayList
- Handling user input with Scanner
- Basic Git and GitHub workflow
